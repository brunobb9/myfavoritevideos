# My Favorite Videos  
A React application that allows users to search YouTube videos using the official YouTube Data API v3 and save their favorite ones. The app includes a responsive card layout, clickable thumbnails, and simple state-based management of a personal favorites list.

---

## Features
- Search YouTube videos in real time  
- Click thumbnails to open videos on YouTube  
- Add videos to a favorites list  
- Remove videos from favorites  
- Favorites stored using React state  
- Responsive card layout  
- Built with Vite for fast development

---

## Tech Stack
- React.js  
- Vite  
- CSS (custom styling)  
- YouTube Data API v3 (REST)
